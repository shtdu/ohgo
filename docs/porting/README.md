# Purpose

The documents in this folder is for porting the Python OpenHarness to Go "ohgo". After the porting, this folder should be removed.

## Documents

- [Porting Guide](GUIDE.md): How to port features from the Python OpenHarness to Go. This is the reference for translating Python patterns into idiomatic Go.
- [Porting Plan](PLAN.md): The plan for porting the Python OpenHarness to Go, by a MVP approach.
